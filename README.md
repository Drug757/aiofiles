# aiofiles project

## 1. Краткое описание инструмента
aiofiles — это библиотека для асинхронной работы с файлами в Python.

## 2. Цель изучения
Понять, как работает асинхронный ввод-вывод и как избежать блокировки программы.

## 3. Описание реализованной задачи
Реализована программа, которая записывает текст в файл и затем читает его асинхронно.

## 4. Инструкция по запуску
1. Установить библиотеку:
   ```
   pip install aiofiles
   ```
2. Запустить:
   ```
   python main.py
   ```
## 5. Общие выводы
Библиотека полезна для асинхронных приложений и помогает ускорить работу программы.
