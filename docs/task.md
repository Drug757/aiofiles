# Техническое задание

## 1. Описание задачи
Необходимо разработать программу, которая демонстрирует асинхронное чтение и запись файлов с использованием библиотеки aiofiles.

## 2. Цель работы
Изучить основы асинхронной работы с файлами и понять, как избежать блокировки программы при вводе-выводе.

## 3. Функциональные требования
Программа должна:
- создать файл  
- записать в него текст  
- прочитать содержимое файла  
- вывести результат в консоль  

## 4. Нефункциональные требования
- использование async/await  
- код должен быть читаемым  
- программа должна выполняться быстро  

## 5. Ограничения и допущения
- используется один файл  
- текст небольшой  
- программа запускается из консоли
